About Liveliness

Liveliness is a cointime economics metric that measures the ratio of coin days destroyed to coin days created. It indicates whether the network is in a HODLing phase or a spending phase.

  • Liveliness = CDD / (CDD + CDC)
  • CDD: Coin Days Destroyed (spending old coins)
  • CDC: Coin Days Created (new coin days accumulating)
  • Range: 0 to 1 (0 = all HODLing, 1 = all spending)
